# Changelog

## 0.1.0

- Initial release.
- Server-driven UI renderer for the bapp_framework backend.
- Manifest/screen/node models with a pluggable node registry.
- Built-in nodes, input nodes, and field widgets.
- Dashboard, form, and settings templates plus list/detail screens.
- Value-conditional rendering (`show_when`) and tag field options.
- Access-driven routing with app/tenant selection, persistence, and 403 reset.
- Overlay rendering: present sheet, dialog, and popover from an icon node.
- Action runner and dispatcher with navigation support.
